PHPEPL fork

A fork of the original PHPEPL built by Mr. Joel Kemp at https://github.com/mrjoelkemp/phpepl.

This version adds support for snippets, a slick GUI, and a convenience function which wraps things in var_dump for you at the last line, if the line starts with a >. So…

// some code here
> $x, $y, $z

…will compile to

// some code here
var_dump($x, $y, $z);

You must omit the semi-colon at the end. And it only works for the last line. Just a handy thing to have.

Snippets are automatically saved in localStorage, you can filter the snippets too. Have fun!
Due credits to Mr. Joel Kemp for the original PHP Eval script.

Made with <3, JavaScript, and CSS3 (Flexbox for the layout), and 2 hours.
